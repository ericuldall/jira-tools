# JIRA Tools

JIRA Tools is a cli tool that allows you to perform actions against your JIRA projects via your command line.


## Getting Started

*INSTALL*
```
npm install -g jira-tools
```

*AUTHENTICATE*
```
jtools auth -i && source ~/.bashrc
```

*RUN A COMMAND*
```
jtools -p [PROJECT] realease get --all
```
